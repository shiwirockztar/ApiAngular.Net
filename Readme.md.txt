Descripción general:
El candidato debe desarrollar una aplicación web que permita a los usuarios buscar películas
y guardarlas en su perfil. La aplicación debe estar basada en Angular para el frontend y .NET
Core 6.0 para el backend. Además, debe utilizar una API pública de películas para obtener datos
de películas. Se debe implementar un sistema de autenticación y una base de datos para
almacenar información de usuarios y las películas guardadas por cada usuario.
Api de películas recomentado, no obligatoria: Getting Started (themoviedb.org)
(https://developer.themoviedb.org/reference/intro/getting-started)
